# r_t_t.py 策略选择配置示例

## 如何控制显示哪些策略

### 方法1：在脚本中配置（推荐）

编辑 `r_t_t.py` 文件顶部的配置区域：

```python
# 策略选择
DEFAULT_STRATEGIES = None  # 显示所有策略
```

修改为：

```python
# 只显示 ga 和 gal
DEFAULT_STRATEGIES = ["ga", "gal"]

# 只显示 ga、gal 和 pretrain
DEFAULT_STRATEGIES = ["ga", "gal", "pretrain"]

# 只显示 pretrain 和 finetuned
DEFAULT_STRATEGIES = ["pretrain", "finetuned"]

# 只显示 ga
DEFAULT_STRATEGIES = ["ga"]
```

然后直接运行：
```bash
python3 r_t_t.py
```

### 方法2：使用命令行参数

```bash
# 只显示 ga 和 gal
python3 r_t_t.py --strategies ga gal

# 只显示 pretrain 和 finetuned
python3 r_t_t.py --strategies pretrain finetuned

# 只显示 ga
python3 r_t_t.py --strategies ga
```

## 常见策略名称

根据数据文件，常见的策略名称包括：
- `ga` - 遗传算法
- `gal` - GAL算法（基于NodeRank）
- `gal_pn` - GAL-PN算法
- `gal_sn` - GAL-SN算法（现在改名为greedy）
- `pretrain` - 预训练模型
- `finetuned` - 微调模型

## 完整配置示例

```python
# ============================================================================
# 配置区域：修改这里的参数来控制脚本行为
# ============================================================================

# 输入文件路径
DEFAULT_INPUT_FILE = "round_2_rt_over_time_data.json"

# 时间轴设置
DEFAULT_START_TIME_STEP = 0      # 起始时间步
DEFAULT_END_TIME_STEP = 1000     # 结束时间步

# 策略选择 - 只显示 ga 和 gal
DEFAULT_STRATEGIES = ["ga", "gal"]

# 显示选项
DEFAULT_USE_ABS = True           # 使用绝对值

# 输出选项
DEFAULT_OUTPUT_FILE = None        # 自动生成文件名

# 图表样式
DEFAULT_FIGSIZE = (14, 8)         # 图表大小
DEFAULT_DPI = 200                 # 分辨率
DEFAULT_MARKER_SIZE = 2           # 标记大小
DEFAULT_LINEWIDTH = 1.5           # 线宽
```

## 运行示例

### 示例1：只显示 ga 和 gal

```python
# 在脚本中设置
DEFAULT_STRATEGIES = ["ga", "gal"]
```

运行：
```bash
python3 r_t_t.py
```

### 示例2：只显示 pretrain 和 finetuned

```python
# 在脚本中设置
DEFAULT_STRATEGIES = ["pretrain", "finetuned"]
```

运行：
```bash
python3 r_t_t.py
```

### 示例3：只显示 ga，时间范围 100-500

```python
# 在脚本中设置
DEFAULT_STRATEGIES = ["ga"]
DEFAULT_START_TIME_STEP = 100
DEFAULT_END_TIME_STEP = 500
```

运行：
```bash
python3 r_t_t.py
```

## 查看可用策略

运行脚本时，会自动显示数据文件中所有可用的策略：

```
配置:
  起始时间步: 0
  结束时间步: 自动（使用数据文件中的max_time_steps）
  可用策略: finetuned, ga, gal, gal_pn, gal_sn, pretrain
  显示策略: 所有策略 (6 个)
  ...
```

如果指定的策略不存在，会显示警告：

```
配置:
  ...
  可用策略: finetuned, ga, gal, gal_pn, gal_sn, pretrain
  警告：以下策略不存在: invalid_strategy
  显示策略: ga, gal
  ...
```
